# my-commander
> 基于commander的自定义命令行工具

### 安装方法
```
# bash
git clone https://github.com/dancdx/my-commander.git
cd my-commander
npm link

```
### 使用说明
> 创建组件（默认样式文件css）
```
simr c App
// simr component App
```  
> 使用sass
```
simr c App --sass
// simr component App --sass
```  
> 使用less
```
simr c App --less
// simr component App --less
```  

> 目前只有创建组件的命令，节省了开发过程中创建组件js和组件css的步骤和通用代码的编写，提高开发效率，其他命令会陆续开发，敬请期待～
